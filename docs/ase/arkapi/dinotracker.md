# 恐竜の位置検索機能
恐竜の位置検索機能はテイム済みの自分のトライブの恐竜の座標を探すことができます

## 使い方
チャット欄(全体チャット)に`/t 検索ワード`と入力すると検索ができます。  
※「/」「t」「空白」はすべて半角で入力してください。

検索ワードには、種族名(例:ティラコレオ)や付けた名前(例:ねこ)などを使用できます。  
また、ティラコレオに「ねこ」と名付けていても、`/t ティラコレオ`で検索した場合にもヒットします。  
名前の一部だけでも検索可能です。

この機能の使用には750ポイントが必要です。 
ポイントについては[こちら](/docs/ase/arkapi/arkshop.md)

検索結果の表示時間は30秒です

## 注意点
- ヒットしなくてもポイントは消費されます。  
- ヒットした恐竜が多すぎる場合、表示しきれないことがあります。  
- フィヨルドなどの別フィールドがあるマップではその座標に行っても距離が離れているように見えます。その場合はアスガルドなど別のフィールドにいる可能性があります。
