# サポートに関して(ASA)

サポートとはバグや重大なミスなどの補填　復旧をするサービスのことです

## サポートできること

基本的にバグは出来る限り対応をします　ミスなどは重大なもののみ対応します。
[こちら](https://discord.com/channels/870642671415337001/1219128674461421700)に沿ってサポートを投稿してください。

主なサポートと注意事項

* アイテムロスト  
何らかの要因でアイテムや恐竜をロストした場合にこちらをご利用ください。死体が埋まったなどもこちらです。  
ただし、洞窟攻略中にマグマや奈落に落下した場合など、明らかにプレイヤーの自己責任によるロストについては対応できません。<br></br>
* サバイバーロスト  
何らかの要因でサバイバーが消えてしまった場合にこちらをご利用ください。<br></br>
* 恐竜回収  
何らかの要因で恐竜が回収できなくなってしまったときにこちらをご利用ください。
地面に埋まってしまった場合でも、サブスクリプション加入者（ラプトル／レックス／ギガノト／キンタン）は、ゲーム内チャットで``/pod``と入力することで回収できる可能性があります。<br></br>
* 撤去依頼  
違反建築・放置恐竜・地面に埋まって自力で回収できない建築物などの撤去依頼はこちらをご利用ください。
詳細な条件はDiscordチャンネル「建築物-生物-撤去依頼-asa」ピン留め投稿や、フォームの「撤去依頼」項目の注意事項を必ずご確認ください。<br></br>
* サバイバー名変更  
管理者からの変更指示を受けた場合や、自発的に名前を変更したい場合にご利用ください。<br></br>
* サブスクリプション関係  
サブスクリプションに関する内容や、ID登録などに関するお問い合わせはこちらをご利用ください。<br></br> 
* トライブ関連  
何らかの要因でトライブから抜けてしまったなどの場合にこちらをご利用ください。<br></br>
* スタック(はまって身動きできない)  
通常の手段で自殺できない場合などにこちらをご利用ください。<br></br>
* 鯖落ち報告  
どちらの鯖が落ちたのかを教えてください。Discordチャンネル「asa-雑談」にて管理人に対するメンションでも代用可。<br></br>
* その他  
上記のいずれにも当てはまらない場合にご利用ください。
ただし、「とりあえずこれでいいか」といった理由で安易に選択することはお控えください。
